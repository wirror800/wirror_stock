
wirror_stock
========
股票股市仿真Discuz插件  
* 全真模拟
* 交易模拟


使用方法：  
================
* 将插件拷贝到Discuz的插件目录
* 在后台应用管理中启用wirror_stock插件
* 配置导航链接
  

DISCUZ版本支持
================
* X2.5 X3.2
